# SentinelAPI
R Sentinel Data Access API

This package is still under development, but most functionality is complete.

Purpose: To download Sentinel satellite imagery (1, 2 or 3) for an AOI. Package allows user to first download all thumbnails and preview images for a geographic region of interest and date range. Full georeferenced rasters are downloaded automatically overnight.
